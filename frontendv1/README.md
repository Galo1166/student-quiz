# FrontendV1

A modern React + TypeScript project with Vite.

## Getting Started

### Install Dependencies
```bash
npm install
```

### Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Preview Build
```bash
npm run preview
```

### Lint
```bash
npm run lint
```

## Technologies

- **React 19.2.0** - Modern UI library
- **TypeScript 5.9** - Type-safe JavaScript
- **Vite 7.3** - Fast build tool & dev server
- **ESLint** - Code quality
- **Tailwind CSS Ready** - Add with: `npm install tailwindcss postcss autoprefixer`

## Project Structure

```
frontendv1/
├── src/
│   ├── main.tsx     - Entry point
│   ├── App.tsx      - Main component
│   ├── App.css      - App styles
│   └── index.css    - Global styles
├── index.html       - HTML template
├── package.json     - Dependencies
├── tsconfig.json    - TypeScript config
└── vite.config.ts   - Vite config
```
